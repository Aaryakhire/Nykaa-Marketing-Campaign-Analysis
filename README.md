# Nykaa-Marketing-Campaign-Analysis

## Description
Analyzed Nykaa marketing campaign data using MySQL, Python, and Power BI to identify high-performing campaigns, measure ROI, calculate conversion metrics, and build interactive dashboards for revenue, audience, and channel performance analysis.

---

## Tools and Technologies Used
- MySQL
- Pandas
- PowerBI

---

## Key Tasks Performed
- Cleaned and transformed Nykaa campaign data.
- Built Python analysis for KPI and trend evaluation.
- Identified campaign, audience, and channel performance patterns.
- Detected inefficient campaigns.
- Designed interactive Power BI dashboard ideas with slicers and advanced visuals.
